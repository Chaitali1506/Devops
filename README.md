# DevOps Practice Repository

This repository is created to practice **Linux commands**, **Git/GitHub basics**, and to understand **DevOps fundamentals** using Ubuntu Server and Git Bash.

---

## Topics Covered

### User Management
- `whoami` – Current user
- `who` – Logged-in users
- `adduser <user>` – Create user
- `passwd <user>` – Set password
- `su <user>` – Switch user
- `exit` – Logout user

---

### Package Management
- `apt update` – Update package list
- `apt upgrade` – Upgrade packages
- `apt install <pkg>` – Install package
- `apt remove <pkg>` – Remove package

---

###  Process & System Management
- `ps`, `ps -ef` – Running processes
- `top` – Live process monitoring
- `kill <pid>` – Kill process
- `df -h` – Disk usage
- `free -m` – Memory usage

---

### Git Basics
- `git init`, `git status`
- `git add`, `git commit`
- `git log`
- `git branch`, `git checkout`

---

### GitHub Commands
- `git remote add origin <url>`
- `git push origin master`
- `git pull origin master`
- `git clone <url>`

---

### DevOps Tools (Basics)
- CI/CD
- Jenkins
- Docker
- Kubernetes
- Ansible
- Terraform
- AWS

---

## Objective
To build strong fundamentals in **DevOps**, **Linux administration**, and **version control** through hands-on practice.
